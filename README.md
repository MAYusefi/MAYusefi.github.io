# Mohammad Ali Yusefi - Portfolio

A modern, responsive personal portfolio website showcasing professional experience, skills, projects, and certifications.

**Live Demo:** [MAYusefi.github.io](https://MAYusefi.github.io/)

## Features

- Responsive design optimized for all devices
- Dark/Light theme with system preference detection
- Progressive Web App (PWA) support
- Performance optimized with lazy loading
- WCAG compliant accessibility
- SEO friendly with meta tags and sitemap
- Bilingual support (English/Dari)

## Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- CSS Custom Properties
- CSS Grid and Flexbox
- Service Worker for offline support
- GitHub Actions for CI/CD

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/MAYusefi/MAYusefi.github.io.git
   ```

2. Open `index.html` in your browser or use a local server
   ```bash
   npx serve .
   ```
